Functions in Python - III

Python Functions are reusable and structured blocks of code designed to perform specific tasks efficiently. They help in organizing a program into smaller, manageable, and modular sections. Functions improve code readability, reduce repetition, and enhance maintainability. By using functions, complex problems can be broken down into simpler parts, making programs easier to understand and debug. In this lab, you will study different types of functions in Python, including user-defined functions, functions with parameters, return values, recursive functions, lambda functions, and functions with variable-length arguments.

Functions in Python – Practice Questions

Below are function-based programming questions with their implementation.

Q1. Write a function to check whether a number is even or odd.

def check_even_odd(num):
    if num % 2 == 0:
        return "Even"
    else:
        return "Odd"

print(check_even_odd(10))

Q2. Write a function to calculate the factorial of a number.

def factorial(n):
    if n == 1:
        return 1
    return n * factorial(n-1)

print(factorial(5))

Q3. Write a function to find the largest of three numbers.

def largest(a, b, c):
    return max(a, b, c)

print(largest(10, 25, 15))

Q4. Write a function that returns the sum of all elements in a list.

def list_sum(lst):
    return sum(lst)

print(list_sum([2,4,6,8]))

Q5. Write a function to check whether a string is a palindrome.

def is_palindrome(text):
    return text == text[::-1]

print(is_palindrome("madam"))

Q6. Write a function using *args to find the average of numbers.

def average(*numbers):
    return sum(numbers)/len(numbers)

print(average(10,20,30))

Q7. Write a lambda function to find the square of a number.

square = lambda x: x*x
print(square(7))

Q8. Write a function that returns multiple values (addition and multiplication).

def operations(a, b):
    return a+b, a*b

add, mul = operations(5,3)
print("Addition:", add)
print("Multiplication:", mul)
