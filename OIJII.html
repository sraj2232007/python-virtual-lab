Conditions in Python - II

Python operators are special symbols that perform operations on variables and values. They are used to manipulate data and variables in expressions. In this lab you will be studying about conditional statements and additional types of operators in Python.

Conditional Statements in Python

Python provides several ways to make decisions:

If Statement

The if statement is used to execute a block of code only if a condition is true.

  • if condition: Executes code if condition is True.

Program:

age = 18

if age >= 18:
    print("You are eligible to vote")

Output:

You are eligible to vote

If-Else Statement

The if-else statement executes one block if a condition is true, and another block if it's false.

  • if condition: Executes if condition is True.
  • else: Executes if condition is False.

Program:

age = 15

if age >= 18:
    print("You are eligible to vote")
else:
    print("You are not eligible to vote")

Output:

You are not eligible to vote

If-Elif-Else Statement

The if-elif-else statement is used to test multiple conditions. The elif stands for "else if".

  • if condition1: Executes if condition1 is True.
  • elif condition2: Executes if condition2 is True.
  • else: Executes if all conditions are False.

Program:

marks = 75

if marks >= 90:
    print("Grade: A")
elif marks >= 80:
    print("Grade: B")
elif marks >= 70:
    print("Grade: C")
else:
    print("Grade: F")

Output:

Grade: C

Nested If Statements

You can have if statements inside if statements. This is used for multiple conditions.

Program:

age = 20
income = 25000

if age >= 18:
    if income >= 20000:
        print("You can apply for a loan")
    else:
        print("Your income is insufficient")
else:
    print("You are not eligible")

Output:

You can apply for a loan

Compound Conditions (and, or, not)

You can combine multiple conditions using logical operators: and, or, and not.

Program:

age = 25
income = 30000
credit_score = 750

# Using 'and' - both conditions must be true
if age >= 18 and income >= 25000:
    print("Eligible for basic loan")

# Using 'or' - at least one condition must be true
if age < 18 or income < 25000:
    print("May need a guarantor")

# Using 'not' - negates the condition
if not (age < 18):
    print("You are an adult")

# Complex conditions
if age >= 18 and income >= 25000 and credit_score >= 700:
    print("Eligible for premium loan")

Output:

Eligible for basic loan
You are an adult
Eligible for premium loan

Dictionary-Based Switch Pattern

Python doesn't have a native switch statement. Instead, you can use a dictionary to map values to actions.

Program:

day = 3

days = {
    1: "Monday",
    2: "Tuesday",
    3: "Wednesday",
    4: "Thursday",
    5: "Friday",
    6: "Saturday",
    7: "Sunday"
}

if day in days:
    print("Day: " + days[day])
else:
    print("Invalid day")

Output:

Day: Wednesday

Match Statement (Python 3.10+)

The match statement is a powerful way to handle multiple conditions. It's similar to switch in other languages.

Program:

grade = "B"

match grade:
    case "A":
        print("Excellent! Great job!")
    case "B":
        print("Good! Well done!")
    case "C":
        print("Satisfactory! Keep trying!")
    case "D":
        print("Fair! Need improvement!")
    case "F":
        print("Failed! Try again!")
    case _:
        print("Invalid grade!")

Output:

Good! Well done!

Bitwise Operators

Bitwise operators perform operations on bits.

  • Bitwise And (&) : It compares each bit of two numbers, and the result is 1 only if both bits are 1, else 0.
  • Bitwise OR (|) : It compares each bit of two numbers, and the result is 1 if at least one bit is 1, else 0.
  • Bitwise XOR (^) : Exclusive OR, and the result is 1 if bits are different, 0 if they are the same.
  • Bitwise NOT (~) : It flips all bits. In Python, ~a = -(a+1).

Program:

a = 5    # Binary: 0101
b = 7    # Binary: 0111

print("a & b = " + str(a & b))  # AND
print("a | b = " + str(a | b))  # OR
print("a ^ b = " + str(a ^ b))  # XOR
print("~a = " + str(~a))        # NOT

Output:

a & b = 5
a | b = 7
a ^ b = 2
~a = -6

Bitwise Shift Operators

Shift operators are used to shift the bits of a number left or right. Left shift multiplies by 2^n, right shift divides by 2^n.

  • Left shift (<<) : This operator shifts all bits to the left. Each left shift multiplies by 2^n.
  • Right shift (>>) : This operator shifts all bits to the right. Each right shift divides by 2^n.

Program:

a = 4   # Binary: 0100

print("Left Shift (a << 2): " + str(a << 2))    # 4 * 2^2 = 16
print("Right Shift (a >> 2): " + str(a >> 2))   # 4 / 2^2 = 1

Output:

Left Shift (a << 2): 16
Right Shift (a >> 2): 1

Ternary Operator

The ternary operator is a shorthand form of the if-else statement. It evaluates a condition and returns one value if true, another if false.

Syntax:

result = value_if_true if condition else value_if_false
  • If condition is true, then it returns value_if_true.
  • If condition is false, then it returns value_if_false.

Program:

a = 10
b = 5

max_num = a if a > b else b
print("The maximum between two numbers is: " + str(max_num))

# Another example
age = 25
status = "Adult" if age >= 18 else "Minor"
print("Status: " + status)

Output:

The maximum between two numbers is: 10
Status: Adult
